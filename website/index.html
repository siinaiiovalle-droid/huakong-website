关于华控

以自主可控的工业自动化与信息化技术
让中国钢铁从"制造"走向"智造"

使命 Mission

以自主可控的工业自动化与信息化技术，让中国钢铁工业从"制造"走向"智造"。我们把对工艺、设备、数据、决策的理解，转化为可交付、可运维、可持续进化的工业软件与控制系统。

愿景 Vision

成为中国钢铁行业最值得信赖的工业自动化与数字化转型合作伙伴；用十年时间，参与并见证中国钢铁工业从世界最大走向世界最强。

价值观 Values

  • 客户至上 · 客户验收前不出厂，稳定运行前不离场
  • 自主创新 · 核心技术不外包，控制系统不依赖国外品牌
  • 精益求精 · 现场调试一次成功率高于 95%
  • 共生共赢 · 与客户、员工、合作伙伴共同成长

战略路线

硅钢起步 · 全厂覆盖 · 走向行业

2026

奠基期

完成武钢硅钢厂 L2/L3/L4 三层系统全栈验收，树立行业样板。

  • 硅钢厂 L2 过程控制系统上线
  • 硅钢厂 L3 MES 一期上线
  • 硅钢厂 L4 ERP 一期上线
2027–2028

复制期

把硅钢样板复制到热轧、冷轧、镀锌等同类产线。

  • 热轧厂、冷轧厂（一期）交付
  • 镀锌厂、冷轧厂（二期）覆盖
  • Huakong-MES Cloud 版本发布
2029–2030

拓展期

拓展至炼铁、炼钢、焦化、烧结等长流程主工序。

  • 炼铁厂、炼钢厂覆盖
  • 焦化厂、烧结厂覆盖
  • 至少 8 大类钢厂完成覆盖
2031–2035

生态期

输出方法论与平台，参与行业标准制定，形成"华控标准"。

  • 集团 BI / 调度中心运行
  • 多基地协同平台上线
  • 参与钢铁自动化行业标准制定
